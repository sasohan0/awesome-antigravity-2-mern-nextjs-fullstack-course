# Course Map

This file is the quick navigation map for the **Awesome Antigravity 2.0 MERN + Next.js Full-Stack Course**.

Use it when you want the fastest view of module order, estimated time, deliverables, and completion standard.

01 NotebookLM
02 Antigravity setup
03 spec-first development
04 architecture
05 SKILL.md mastery
08 frontend
09 backend
10 integration
11 testing
12 deployment
13 final assignment
17 security

```txt

```

## Required proof per core module

Each core module should produce:

```txt
1. Module deliverables
2. NotebookLM study evidence
3. Practice task evidence
4. Quiz answers
5. Reflection
```

Final project should produce:

```txt
1. GitHub repository
2. Live frontend
3. Backend health URL
4. Demo credentials
5. README
6. Docs folder
7. QA reports
8. Deployment report
9. Portfolio case study
```

## Completion standard

A student completes the course only when they can say:

```txt
I can plan, build, test, deploy, document, secure, and explain a MERN/Next.js project using Antigravity responsibly.
```
