# Course Resource Index and Update Log

**File Name:** `18-course-resource-index-and-update-log.md`  
**Purpose:** Central resource hub for the Antigravity 2.0 MERN + Next.js Full-Stack Course.  
**Use:** Keep this file updated when docs, tools, videos, codelabs, or course modules change.

## Why this file exists

Antigravity, NotebookLM, Next.js, Vite, Tailwind, React Router, deployment platforms, and testing tools change over time.

This file gives students and mentors one place to check:

```txt
Official links
Video search terms
Module resource mapping
Version notes
Update log
Maintenance checklist
```

When a resource breaks or a tool updates, update this file first.
